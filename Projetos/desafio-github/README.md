# GitHub Users

### Create project
```bash
yarn create vite <project-name> --template react-ts
```
### Install dependencies
```bash
yarn
```

### Install React Router DOM
```bash
yarn add react-router-dom@6.4.1 @types/react-router-dom@5.3.3
```

### start project
```bash
yarn dev
```
